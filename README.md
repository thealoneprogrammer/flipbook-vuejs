# flipbook
Flipbook effect using vue js

## take a demo below

https://thealoneprogrammer.github.io/flipbook-vuejs/


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
